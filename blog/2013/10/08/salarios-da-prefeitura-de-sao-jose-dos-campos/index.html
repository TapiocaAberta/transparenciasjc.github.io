Salários Da Prefeitura De São José Dos Campos.

Oct 8th, 2013 4:12 pm

Você sabe quantas folhas de pagamento a prefeitura emite? você sabe qual é o maior salário que a prefeitura paga? você sabia que a prefeitura publica estes dados e que você pode ter acesso a isto? Neste artigo vamos analisar estes dados e acompanhar de perto os pagamentos dos últimos meses.

A prefeitura de São José dos Campos publica, através do portal de transparência, diversos dados sobre finanças, legislações entre outros. Estes dados, porém, são publicados em formato txt que é um arquivo de texto simples formatado em CSV.

O grande problema da formatação que eles utilizam é que ela dificulta a análise destes dados, e consequentemente a modificação do documento para que ele torne-se mais visível por humanos.

Porém, é possível converter estes dados para formatos mais legíveis tais como JSON, XML, entre outros, possibilitando assim a criação de formas de análise sobre estes dados. Foi o que eu fiz, converti estes dados para formatos legíveis por máquina, e deixei eles disponíveis aqui para que qualquer programador ou curioso possa analisar.

Por exemplo, os salários estão disponíveis neste link porém, veja a forma com que ele é mostrado ao usuário:

[caption id=“attachment_29” align=“aligncenter” width=“300”] Salários no site da prefeitura[/caption]

é uma tabela tosca da qual não se pode fazer filtros, não se pode comparar cada um dos registros, não se pode analisar com períodos anteriores, etc. o botão exportar nos fornece o dado da seguinte forma:

ANO;MÊS;ITEM;CARGO;FUNÇÃO GRATIFICADA;TEMPO DE SERVIÇO (ANOS);REMUNERAÇÃO DO MÊS;VERBA EVENTUAL
2013;08;1;ADMINISTRADOR DISTRITAL ;;0;2.241,68;8.598,78
2013;08;2;ADMINISTRADOR DISTRITAL ;;23;7.996,82;0,00

não, isto não é legal. Mas já é alguma coisa. Pegando estes dados e transformando ele em um JSON por exemplo, ele fica da seguinte maneira:

{
    "ANO": 2013,
    "MÊS": 07,
    "ITEM": 1,
    "CARGO": "ADMINISTRADOR DISTRITAL ",
    "FUNÇÃO GRATIFICADA": "",
    "TEMPO DE SERVIÇO (ANOS)": 0,
    "REMUNERAÇÃO DO MÊS": "6.054,23",
    "VERBA EVENTUAL": "0,00"
},
{
    "ANO": 2013,
    "MÊS": 07,
    "ITEM": 2,
    "CARGO": "ADMINISTRADOR DISTRITAL ",
    "FUNÇÃO GRATIFICADA": "",
    "TEMPO DE SERVIÇO (ANOS)": 23,
    "REMUNERAÇÃO DO MÊS": "7.996,82",
    "VERBA EVENTUAL": "0,00"
}

Isto é um pouco mais legível, mas a forma ideal da publicação dos dados seria através de um web service, que são serviços que permitem com que o dado seja filtrada diretamente na requisição, isso sim é que é seria transparência de verdade.

Veja esta app que foi criada a partir dos dados disponíveis no portal de transparência, agora é possível visualizarmos de maneira amigável.

É possível também analisarmos estes dados em uma tabela comum no google docs disponível logo abaixo. Através dela, podemos fazer filtro de quem ganha mais, quem ganha menos, qual a profissão, quantidade de folhas de pagamento.

Agora sim já temos uma evolução.
